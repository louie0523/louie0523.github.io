2D 플랫포머 전투 시스템 구현

2026년 1월 10일 Unity C# Game Development

개요

이번 주에는 2D 플랫포머 게임의 전투 시스템 구현에 집중했습니다. 플레이어와 적 사이의 상호작용을 정교하게 만들고, 히트박스의 정밀도를 높이는 작업을 진행했습니다.

구현 내용

전투 시스템의 핵심 요소들을 단계별로 구현했습니다:

  • 히트박스 시스템: BoxCollider2D를 활용한 정밀한 충돌 감지
  • 공격 범위 감지: 플레이어의 공격 범위 내 적 탐지 로직
  • 데미지 처리: 피격 시 체력 감소 및 넉백 효과
  • 애니메이션 연동: 공격 모션과 히트박스 활성화 타이밍 동기화

주요 코드 구조

void DetectEnemiesInRange() {
    Collider2D[] hits = Physics2D.OverlapBoxAll(
        attackPoint.position, 
        attackRange, 
        0f, 
        enemyLayer
    );
    
    foreach (Collider2D enemy in hits) {
        enemy.GetComponent<EnemyHealth>().TakeDamage(attackDamage);
    }
}

발생한 문제와 해결

개발 중 여러 에지 케이스를 발견했습니다. 특히 플레이어가 적과 겹쳐 있을 때 공격이 감지되지 않는 문제가 있었습니다. 이를 해결하기 위해 충돌 레이어를 재구성하고, 히트박스의 위치를 조정했습니다.
